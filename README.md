 **Live Demo:**

##  Project Description
This project focuses on predicting the risk of heart disease using Machine Learning techniques. It covers the complete ML pipeline from data analysis to model deployment.

## Workflow

### 1. Data Collection
- Used a heart disease dataset for analysis and prediction.

### 2. Exploratory Data Analysis (EDA)
- Analyzed data distribution and relationships between features  
- Identified important patterns affecting heart disease  

### 3. Data Cleaning & Preprocessing
- Handled missing values  
- Encoded categorical variables  
- Scaled numerical features  

### 4. Model Building
- Trained multiple Machine Learning models  
- Compared their performance using accuracy  

### 5. Model Selection
- Logistic Regression gave the best accuracy  
- Selected it as the final model  

### 6. Prediction
- Used the trained model to predict heart disease risk on new input data  

### 7. Deployment
- Built an interactive web application using Streamlit  
- Users can input health details and get real-time predictions  


##  Features
- User-friendly interface

- Clean and interactive UI
